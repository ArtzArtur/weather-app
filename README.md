# Weather App
> Weather app made in ReactJS and Tailwind CSS for training purposes
> Live demo [_here_](https://artzartur.github.io/weather-app/).


## General Information
- Weather application with search engine, geolocation, country informations following days forecast.  
Custom hooks made in different ways also for training purposes


## Technologies Used
- React Js 18
- React Router 6
- Tailwind CSS 3

## Features
List the ready features here:
- Search engine
- Geolocation
- Actual and forecast for following days
- Information about countries
